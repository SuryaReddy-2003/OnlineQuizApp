# Online Quiz Application (Java + SQLite)

## Description
This is a console-based Online Quiz Application developed using Java and SQLite.  
It allows an Admin to create quizzes and add questions, and Users to attempt quizzes and view scores.

## Features

### Admin
- Login as admin  
- Create quizzes  
- Add multiple questions  

### User
- Login as user  
- View available quizzes  
- Attempt quiz  
- View final score
